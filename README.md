# Python Voting System

## About

A simple CLI Voting System built using Python.
This application allows users to add candidates, cast votes, view vote results, and determine the winner.

---

## ✨ Features

✔️ Add candidates
✔️ Cast votes
✔️ View vote count
✔️ Winner detection
✔️ Case-insensitive voting support

---

## 🛠️ Tech Used

* Python
* Object-Oriented Programming (OOP)
* Dictionaries

---

## 🚀 How to Run

1. Download the project
2. Open the Python file
3. Run the program
4. Choose options from the menu
5. Manage candidates and voting
